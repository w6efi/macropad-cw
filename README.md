Code and detailed instructions associated with QST article submitted 2025-06-25 and accepted for publication.

Program handles a programmable macropad (small keyboard with open-source QMK firmware) to issue my most
frequently-used functions on my radio by sending CAT commands.

This is similar to a contest keypad, but this one is programmable with whatever functions you want.

Since the time the article was written, I have updated the program and documentation here; however previous versions
can be viewed in the earlier commits.

The main point is that you can make your own control surface if you want more buttons and knobs for your radio and you are not
limited by commercial products.
